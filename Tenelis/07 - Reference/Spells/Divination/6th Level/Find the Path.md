# Find the Path

This spell allows you to find the shortest, most direct physical route to a specific fixed location that you are familiar with on the same plane of existence. The spell fails if you choose a destination on another plane of existence, a moving destination, or a nonspecific destination. For the duration, as long as you are on the same plane as the destination, you know how far it is and in what direction it lies.
