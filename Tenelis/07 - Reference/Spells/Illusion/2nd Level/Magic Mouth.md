# Magic Mouth

You implant a message within an object in range, a message that is uttered when a trigger condition is met. Choose an object that you can see and that isn't being worn or carried by another creature. Then speak the message, which must be 25 words or fewer, and determine the trigger condition. When that condition is met, a magical mouth appears on the object and recites the message in your voice at the same volume you spoke. Each casting lasts until dispelled or the object is destroyed.
