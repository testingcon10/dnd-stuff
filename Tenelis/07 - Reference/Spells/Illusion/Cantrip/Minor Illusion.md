# Minor Illusion

You create a sound or an image of an object within range that lasts for the duration. The illusion also ends if you dismiss it or cast this spell again. If you create a sound, it can be any sound from a whisper to a scream. An image must fit within a 5-foot cube and can't create sensory effects. A creature can use an action to make an Intelligence (Investigation) check against your spell save DC to determine the illusion.
