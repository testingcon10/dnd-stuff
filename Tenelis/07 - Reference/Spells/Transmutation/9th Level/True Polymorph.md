# True Polymorph

You choose one creature or nonmagical object you can see within range and transform it into a new form. A creature can be transformed into another creature or into an object, and an object can be transformed into a creature. The transformation lasts for the duration, or until the target drops to 0 hit points or dies, and if you concentrate on this spell for the full duration, the transformation becomes permanent until dispelled.
