# Druid

A priest of the Old Faith, wielding the powers of nature and adopting animal forms. Druids use Wild Shape to transform into beasts and command a versatile spell list focused on nature, healing, and battlefield control.

| Property | Value |
|----------|-------|
| **Hit Die** | d8 |
| **Primary Ability** | Wisdom |
| **Saving Throws** | Intelligence, Wisdom |
| **Armor** | Light, Medium, Shields (no metal) |
| **Weapons** | Clubs, Daggers, Darts, Javelins, Maces, Quarterstaffs, Scimitars, Sickles, Slings, Spears |
