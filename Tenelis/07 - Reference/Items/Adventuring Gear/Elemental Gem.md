# Elemental Gem

This gem contains a mote of elemental energy. When you use an action to break the gem, an elemental is summoned as if you had cast the [[Conjure Elemental|conjure elemental]] spell, and the gem's magic is lost. The type of gem determines the elemental summoned: blue sapphire for air, yellow diamond for earth, red corundum for fire, or emerald for water.
